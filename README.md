### Requirements:
- [Node.js LTS](https://nodejs.org/en/download/)
- [nvm (optional)](https://github.com/nvm-sh/nvm)

```bash
nvm use
npm i
npm run presentation --name=intro # name should be the mdx file name inside /classes
```
